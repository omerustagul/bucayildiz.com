Standard header for homepage sections — gold kicker line + uppercase condensed title, with an optional right-side action.

```jsx
<SectionHeading kicker="Kulüpten" title="Son Haberler"
  action={<Button variant="secondary" size="sm">Tümü</Button>} />
```

Use `onDark` over navy backgrounds, `align="center"` for hero/section intros.
