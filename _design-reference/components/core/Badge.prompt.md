Compact label / status pill. Square-ish corners, uppercase by default.

```jsx
<Badge tone="live" dot>Canlı</Badge>
<Badge tone="gold">Yeni</Badge>
<Badge tone="navy">U-17</Badge>
<Badge tone="neutral">Haber</Badge>
```

Tones: `navy` `gold` `neutral` `outline` `live` `success` `on-navy`. `dot` adds a status dot (pulses for `live`). Requires the `byPulse` keyframe (shipped in card/kit styles).
