Icon-only square button — social media links beside the logo, carousel arrows, compact nav controls.

```jsx
<IconButton label="Instagram" variant="on-navy"><i data-lucide="instagram"></i></IconButton>
<IconButton label="Sonraki" variant="outline"><i data-lucide="chevron-right"></i></IconButton>
```

Variants: `solid` `outline` `ghost` `on-navy` `gold`. Sizes `sm`/`md`/`lg`. Pass `round` for a circular shape. Always provide `label`.
