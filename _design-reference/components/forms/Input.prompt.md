Labeled text field used across the başvuru (application) form and panel login.

```jsx
<Input label="Ad Soyad" placeholder="Öğrencinin adı" required />
<Input label="E-posta" type="email" error="Geçerli bir e-posta girin" />
```

Props: `label`, `hint`, `error`, `leftIcon`, `required`, plus all native input attrs. Gold focus ring; red border on `error`.
